# 小旅人 IP

## 角色定义

小旅人是日式手绘正文配图的默认视觉角色。

小旅人不是吉祥物，不是萌系角色，不是插画主角光环，而是正在认真整理信息、穿过生活场景、照看一个小系统的安静执行者。

## 外形

- 小小的人形角色，细黑线条绘制。
- 圆脸或简单椭圆脸，表情很淡，可以没有五官或只有点状眼睛。
- 简单外套、围巾、帆布包、小帽子或雨伞可选，但不要复杂服装。
- 身体比例朴素，不要大眼萌系，不要夸张 Q 版。
- 可以坐在书桌边、站在窗边、走在纸页地图上、撑伞、搬便签、整理抽屉。
- 角色尺寸不宜过大，通常是画面结构的一部分。

## 性格

- 安静、认真、有耐心。
- 像一个温柔但不煽情的观察者。
- 有一点笨拙和轻微幽默，但不卖萌。
- 更像在整理一段经验，而不是表演。

## 常见职责

让小旅人承担核心动作：

- 把散乱便签夹进手账。
- 在桌面上分拣纸条和小物件。
- 撑伞挡住压力或噪音。
- 在车站月台等待下一班车。
- 把概念放进便当格。
- 从抽屉里取出被遗忘的问题。
- 在窗边观察内外两个状态。
- 沿着纸上小路走过几个节点。
- 给植物浇水，表达长期积累。
- 把线团慢慢绕回线轴。
- 给小房间里的流程开灯、关门、通风。
- 在杯子、书、便签、药盒、日历之间建立关系。

## 禁止

- 不要画成二次元角色。
- 不要画成儿童绘本主角。
- 不要大眼、夸张表情、闪亮眼睛。
- 不要复杂服装、幻想装备、偶像感。
- 不要让小旅人只是站在角落看。
- 不要让小旅人抢走结构表达。
- 不要把角色画得太精致、太商业、太可爱。

## 判断标准

如果去掉小旅人，图的核心隐喻还能完全成立，说明小旅人太装饰了；要重写提示词，让小旅人成为动作主体。
