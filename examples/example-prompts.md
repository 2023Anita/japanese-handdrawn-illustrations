# Example Prompts

## Generate illustrations for a Chinese article

```text
Use $japanese-handdrawn-illustrations to design and generate 4 Japanese hand-drawn illustrations for this Chinese article.
Focus on the core cognitive anchors, not one image per paragraph.
```

## Create a shot list only

```text
Use $japanese-handdrawn-illustrations to analyze where this note needs illustrations.
Only give me a shot list first. Do not generate images yet.
```

## Medical education explainer

```text
Use $japanese-handdrawn-illustrations to create 3 gentle article illustrations for this Chinese medical education handout.
Keep the mood calm, clear, and not childish.
```

## Knowledge workflow article

```text
Use $japanese-handdrawn-illustrations to generate 5 illustrations for this article about turning scattered notes into reusable knowledge.
Prefer desktop organizing, notebook map, and umbrella-protection metaphors.
```

## Refine an image that drifted into anime

```text
Use $japanese-handdrawn-illustrations to revise this image.
Keep the same core idea, but remove anime, manga, big-eyed, cute mascot, and commercial poster qualities.
Make it quiet Japanese editorial notebook sketch style.
```
